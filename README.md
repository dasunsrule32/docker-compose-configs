# Docker Compose Configurations

Docker compose configs for popular applications. Compose files in the `dockge/` directory are designed to work with [Dockge](https://github.com/louislam/dockge). Compose files in the `uncloud/` directory are designed to work with [Uncloud](https://uncloud.run/docs/). 

## How to use
### Dockge

See `README.md` in `dockge/` directory.

### Uncloud

See `README.md` in `uncloud/` directory.
